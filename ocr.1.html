NAME

     ocr – macOS optical character recognition via the command line

SYNOPSIS

     ocr [-hv]

DESCRIPTION

     ocr is a command line interface for the built-in optical character
     recognition capabilities in macOS.

     The following flags are supported:

     -h --help
              Print help and exit.

     -v --version
              Print program name and version, and exit.

     Returns 0 on success and greater than 0 on error or failure.
